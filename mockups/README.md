# Cleve portfolio mockups

Static HTML previews for the portfolio-forward site direction.

## View locally

**Not on GitHub Pages yet** — `strangesongs.github.io/mockups/` returns 404 until you commit and push this folder.

From the repo root (`strangesongs.github.io/`):

```bash
npm run serve
```

Open **http://localhost:8090/mockups/**

Port **8090** avoids conflicting with other local apps on 8080 (e.g. fruit for all).

## Or open files directly

Double-click `index.html` in this folder. Styles are self-contained in `mockups.css` (no server required).
